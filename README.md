# 災害マップ用API 
## 概要
11/18,19の2日間のハッカソンで作成しました。
主な機能は以下の2つになります。
- 避難所の位置データを返す機能
- ユーザが地図上にコメントを残すことができる機能

避難所場所のデータベース(現在は東京都のみ)を持っており、API経由でフロントに情報を送ることができます。
地図情報と組み合わせて利用することを想定しています。
また、ユーザは任意の位置情報に対してコメントを追加できます。
災害時にどこの場所がどのような状況かをテキストデータで記録できます。
